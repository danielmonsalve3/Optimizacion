Proyecto Optimización 

Optimización Numérica

Este repositorio contiene la implementación de diferentes métodos de optimización numérica utilizando Python. Los algoritmos permiten encontrar máximos o mínimos de funciones matemáticas asi como sus raíces mediante métodos numericos

🐍 Contenido del repositorio

En este repositorio se implementan métodos como:

📈 Método de Bisección
🔍 Método de Búsqueda Aleatoria
🧠 Método de falsa posicion
💻 Interpolación Parabólica
📚 Métodos de newton
⭐ Método de la razon dorada
🚀 Visualización de resultados
⚙️ Tablas de iteraciones
📊Graficas

Uso
El programa permite:

Ingresar una función matemática
Elegir si desea maximizar o minimizar
Ejecutar el método de optimización
Mostrar tabla de iteraciones
Mostrar el resultado final
Graficar la función

Ejemplo de función:
🎯 x**2 - 4*x + 5
